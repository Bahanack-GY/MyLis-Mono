## EMPLOYÉ

| Fonctionnalité | Description |
|---|---|
| Tableau de bord | Vue personnelle des tâches, projets et réunions à venir |
| Gestion des tâches | Créer, modifier, commenter et suivre ses propres tâches avec sous-tâches et pièces jointes |
| Planning hebdomadaire | Calendrier de la semaine avec glisser-déposer des tâches |
| Projets | Consulter les projets auxquels l'employé est assigné |
| Tickets de support | Créer et suivre des demandes de support (bug, fonctionnalité, question) |
| Réunions | Consulter et participer aux réunions, voir les comptes-rendus |
| Demandes | Soumettre des demandes (congé, heures supplémentaires, remboursement) |
| Notes de frais | Soumettre et suivre des demandes de remboursement de frais professionnels |
| Documents | Déposer et consulter des documents partagés (contrat, SRS, design…) |
| Formations | Enregistrer et suivre ses formations et certifications |
| Sanctions | Consulter ses propres dossiers disciplinaires |
| Fiches de paie | Télécharger et consulter ses bulletins de salaire |
| Messagerie | Envoyer et recevoir des messages avec ses collègues |
| Profil | Mettre à jour ses informations personnelles et son avatar |
| Notifications | Recevoir et gérer les notifications en temps réel |

## MANAGER

Toutes les fonctionnalités Employé, plus :

| Fonctionnalité | Description |
|---|---|
| Gestion des employés | Créer, modifier, supprimer des employés ; assigner des rôles et des départements |
| Gestion des départements | Créer et gérer les départements, assigner le chef de département |
| Gestion des clients | Base de données clients avec historique, santé client, statuts et conversions |
| Gestion des projets | Créer et suivre les projets : équipe, budget, statut, avancement |
| Administration des tâches | Gérer toutes les tâches de l'organisation, vue Gantt, nature de tâche |
| Facturation | Créer, envoyer et suivre les factures clients ; statuts CRÉÉE, ENVOYÉE, PAYÉE |
| Dépenses | Saisir et catégoriser les dépenses opérationnelles par département |
| Tickets (admin) | Assigner, prioriser et clore les tickets de support |
| Réunions (admin) | Planifier, conduire et rédiger les comptes-rendus avec PV exportable PDF |
| Validation des demandes | Approuver ou rejeter les demandes des employés |
| Documents (admin) | Gérer l'arborescence documentaire et les droits d'accès |
| Validation des notes de frais | Approuver ou rejeter les notes de frais avec consultation des justificatifs |
| Journal d'activité | Audit complet de toutes les actions du système |
| Intégration WhatsApp | Envoyer des messages groupés via WhatsApp |
| Rapports | Tableaux de bord analytiques et exports PDF/CSV |

## COMMERCIAL

Toutes les fonctionnalités Employé, plus :

| Fonctionnalité | Description |
|---|---|
| Tableau de bord commercial | KPIs de vente : leads, taux de conversion, revenus, objectifs |
| Base de leads | Créer et gérer les prospects avec priorité HOT/WARM/COLD et statuts pipeline |
| Pipeline de vente | Vue Kanban des opportunités : Prospection → Qualification → Proposition → Négociation → Closing |
| Suivi des leads | Enregistrer les activités (visite, appel, email, démo) et planifier les prochaines actions |
| Suivi des clients | Gérer les clients existants, rappels de paiement, santé client |
| Conversion lead → client | Convertir un lead qualifié en client avec création de facture et projet |
